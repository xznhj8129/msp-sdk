# Quarantined generator experiments

This directory contains superseded generator experiments, stale duplicate
headers, and old test binaries. It is not a source directory and nothing here
is used by `build_c.sh`.

Files preserve their former repository-relative paths so their provenance is
obvious. The quarantine contents are ignored by Git; this README is the only
tracked file in `.old/`.

Quarantined paths:

* `generator/cgen_test/`
* `generator/get_all_inav_enums_h.py`
* `generator/make_min_header.py`
* `generator/new_get_inav_enums.py`
* `generator/__pycache__/`
* `junkpile/`
